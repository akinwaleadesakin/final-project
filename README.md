# Final Project - N215

## Overview
This is a multi-page website built with HTML, CSS, and JavaScript using the Module Pattern for content management.

## Live Link
(https://in-info-web4.luddy.indianapolis.iu.edu/~aadesaki/n215/final-project/)

## Pages
- **Home** - Hero section, mission statement, and upcoming events
- **About** - Company history and information
- **Contact** - Contact form with email submission
- **Blog** - Blog posts with individual article pages

## Features
- Dynamic content management using JavaScript modules
- Responsive design
- Contact form functionality
- Event listing system
- Blog post management

## Technologies Used
- HTML5
- CSS3
- JavaScript (Module Pattern)
- jQuery

## File Structure
```
final-project/
├── index.html
├── model.js          (Content management)
├── script.js         (Event handlers)
├── style.css         (Styling)
├── images/           (Project images)
└── README.md
```

## How to Use
1. Open `index.html` in a web browser
2. Click navigation links to switch between pages
3. Fill out the contact form to send messages
4. View blog posts and upcoming events
